# Marketing App (`kbteutonia.de`)

This is the public marketing website.

## Local development

```bash
npm run dev --workspace @teutonia/marketing
```

## Vercel

- Project root directory: `apps/marketing`
- Production domain: `kbteutonia.de`
- Required env: `NEXT_PUBLIC_SITE_URL`

## Ownership boundary

- Keep public routes and public-facing content here.
- Do not add auth/session APIs here.
